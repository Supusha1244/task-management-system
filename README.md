# Task Management System
## 📖 About the Project

This is a full-stack Task Management System that allows users to manage their daily tasks efficiently.  
The application provides secure authentication, task creation, updating, deletion, and status tracking.

It is built using modern web technologies and follows a clean and scalable architecture.

---

## 🚀 Features

- 🔐 User Authentication (Register & Login with JWT)
- ➕ Add new tasks
- 📝 Edit existing tasks
- ❌ Delete tasks
- ✅ Mark tasks as completed / pending
- 🔍 Search tasks
- 🎯 Filter tasks (All / Completed / Pending)
- 🔔 Toast notifications for actions
- 📱 Responsive design (Mobile + Desktop)
- 🎨 Beautiful UI with cards and colors
